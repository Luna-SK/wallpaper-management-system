# 项目规划

## 目标

`wzut-wallpaper-manager` 是图片管理系统的新实现。系统面向图片资产管理，支持上传、分类、标签、检索、预览、权限控制、审计日志、统计和备份。

一期把纺织瑕疵图片作为内置业务场景，但不把系统限制为瑕疵管理平台。图片通过分类和分类下标签管理，后续可以扩展到样张、材质库、风格库、教学数据集等场景。

## 技术栈

- 后端：Spring Boot 4.0.x、Java 25 LTS、Maven、Spring Security、Spring Data JPA、Liquibase。
- 前端：Vue 3、TypeScript、Vite、Pinia、Vue Router、Element Plus。
- 存储：RustFS，通过 S3 兼容接口访问。
- 数据库：MySQL 8.4 LTS。
- 缓存：Redis。
- 部署：单机 Docker Compose。

## 一期交付

- 用户注册、登录、密码重置、个人信息修改。
- 四角色权限模型：系统管理员、数据管理员、标签编辑人员、普通浏览用户。
- 图片上传、批量上传、原图保留、预览图、缩略图。
- 分类、分类下标签，一期内置纺织瑕疵分类及 35 个标签。
- 关键词、分类、标签、上传者、时间范围检索。
- 鉴权访问、原图下载、操作日志、访问日志。
- 图片数量、上传量、浏览量、下载量、存储占用统计。
- Docker 部署、空库初始化、备份脚本。

## 非一期范围

- 完整在线裁剪、旋转、批注和复杂图片编辑器。
- 评论、点赞、反馈工单。
- 相似图检索、AI 自动识别、模型训练工作流。
- 多机高可用部署。
