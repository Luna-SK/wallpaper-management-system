# 开发需求清单

本文按当前 `main` 分支实现状态维护。已完成项使用 `[x]`，待开发或后续增强项使用 `[ ]`。

## 用户管理模块

- [x] 用户注册。
- [x] 用户登录。
- [x] 个人信息修改。
- [x] 用户自助修改密码。
- [x] 管理员重置用户密码。
- [ ] 邮件找回密码。
- [x] 基于角色为用户分配权限。
- [x] 按权限控制图片、分类、标签、用户、角色、日志、设置等资源操作。
- [x] 用户支持停用、启用和彻底删除。
- [x] 角色支持停用、启用、彻底删除和权限配置。

## 图片上传与管理模块

- [x] 支持单张上传墙布图片。
- [x] 支持批量上传墙布图片。
- [x] 上传前校验文件类型、大小和权限。
- [x] 图片分类表示主归属，一张图片最多选择一个分类。
- [x] 标签通过标签组组织，标签组表示描述维度，例如瑕疵、风格、颜色、材质。
- [x] 图片可选择多个任意启用标签。
- [x] 一期保留 `纺织瑕疵` 分类。
- [x] 原 35 个瑕疵标签归入 `瑕疵` 标签组。
- [x] 图片编辑支持元数据编辑。
- [x] 图片编辑支持分类调整。
- [x] 图片编辑支持标签调整。
- [x] 在线裁剪。
- [x] 在线旋转。
- [x] 在线缩放。
- [x] 水印能力。

## 图片存储与备份模块

- [x] 图片对象存储在 RustFS 或兼容 S3 的对象存储中。
- [x] 上传后保留原图。
- [x] 上传后生成缩略图。
- [x] 上传后生成标准预览图。
- [x] 上传后生成高清预览图。
- [x] 数据库通过 Docker MySQL 备份。
- [x] 对象存储备份策略在部署文档中形成可执行说明。

## 图片检索与浏览模块

- [x] 支持关键词搜索。
- [x] 支持分类筛选。
- [x] 支持标签筛选。
- [x] 支持图片预览。
- [x] 标签筛选不依赖分类。
- [x] 标签选择展示标签组信息。
- [x] 标签筛选器按标签组分组展示和选择。
- [x] 图片详情基础展示大图。
- [x] 图片详情基础展示分类。
- [x] 图片详情基础展示标签。
- [x] 图片详情基础展示文件名。
- [x] 图片详情完整展示尺寸、文件大小、MIME、上传时间、访问量、下载量等信息。
- [x] 预览或下载后，前端列表和详情中的访问量、下载量即时刷新。

## 权限控制与安全模块

- [x] 按角色限制图片访问和管理操作。
- [x] 操作日志记录登录。
- [x] 操作日志记录上传。
- [x] 操作日志记录编辑。
- [x] 操作日志记录下载。
- [x] 操作日志记录删除。
- [x] 操作日志记录分类标签维护。
- [x] 操作日志记录系统设置变更。
- [x] 敏感配置不提交到版本库。
- [x] 账号密码等敏感数据使用哈希存储。
- [x] 图片版权保护以鉴权下载为基础。
- [x] 图片版权保护以访问审计为基础。
- [x] 图片版权保护以预览图控制为基础。
- [x] 图片水印版权保护。
- [ ] 登录空闲超时自动退出。
- [ ] 绝对会话时长控制。

- [x] 统计图片数量。
- [x] 统计上传量。
- [x] 统计访问量。
