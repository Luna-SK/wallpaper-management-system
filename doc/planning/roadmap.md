# Roadmap

## M0 基线

- 创建项目结构、文档结构、后端与前端脚手架。
- 完成 Docker Compose 配置。
- 建立 Liquibase 空库初始化基线。
- 提供导入器骨架。

## M1 核心数据闭环

- 实现真实账号注册、登录、刷新令牌、退出登录。
- 实现用户、动态角色、权限配置管理。
- 实现 RustFS bucket 初始化和图片对象写入。
- 实现图片元数据入库、SHA-256 去重、预览图和缩略图生成。

## M2 图片管理闭环

- 实现图片库分页、搜索、筛选、详情、删除、恢复。
- 实现分类、分类下标签维护。
- 实现纺织瑕疵目录导入和标签绑定。
- 实现访问日志、下载授权和预览质量控制。

## M3 验收与部署

- 完成后端集成测试、前端单测和关键 E2E 测试。
- 完成空库启动验收。
- 完成备份恢复演练。
- 补齐部署文档和发布检查清单。
