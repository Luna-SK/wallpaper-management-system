# 图片导入计划

## Input

导入工具读取 `DATA_DIR` 下的图片目录。当前数据集使用 `0` 到 `20` 的文件夹编号组织纺织瑕疵图片。新系统把这些编号映射为 `纺织瑕疵` 分类和 `瑕疵` 标签组下的标签名，不把编号写死在图片表中。

## Flow

1. 导入器扫描目录和图片扩展名。
2. 导入器根据文件夹编号得到一个或多个标签名。
3. 导入器调用 `POST /api/imports/images`，提交文件、来源路径、分类编码、分类名称、标签组编码和标签名。
4. 后端完成文件校验、去重、RustFS 写入、元数据入库和标签绑定。
5. 导入器输出成功、失败和跳过数量。

## Acceptance

- 默认 dry run 不上传文件。
- 重复文件通过 SHA-256 幂等处理。
- 任一导入失败不影响后续文件。
- 导入结果包含可追踪的源文件路径。
